_pBEAM.Beam.criticalBucklingLoads¶

Beam.criticalBucklingLoads()¶

Estimates the minimum critical (global) buckling loads due to axial loading in addition to any existing input loads.

Returns :

Pcr_x : float (N)

critical axial load for buckling in the x-direction

Pcr_y : float (N)

critical axial load for buckling in the y-direction
